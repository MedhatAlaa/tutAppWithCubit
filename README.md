# TutApp 

Tut App  Using Cubit As State management with Dark Mode

 # Features used in app: 
- Rest full (Api)
- Clean Architecture Design Pattern
- MVVM - Model - View - View Model Pattern
- Dependency injection
- Application Layer
- Domain layer
- Data layer
- Presentation layer
- State renderer 
- Mappers
- Data sources
